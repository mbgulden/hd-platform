Free Human Design Chart Widget

Discover your Type, Strategy, Authority, and more — instantly.

🔮 No sign-up required

📦 Embed on Your Site

Drop this one script tag and a placeholder div anywhere on your page. The widget auto-discovers .hde-chart-widget elements and initializes itself. 

<!-- 1. Include the widget script -->
<script src="https://humandesignengine.com/widget.js"></script>

<!-- 2. Place a widget container -->
<div class="hde-chart-widget"></div>

⚙️ Manual Initialization

// Target any element by selector
HDEWidget.init('#my-widget', {
  apiUrl: 'https://api.humandesignengine.com'
});
